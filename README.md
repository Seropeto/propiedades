# 🏠 Toxiro Propiedades — Publicación Automatizada de Propiedades con IA

> Genera publicaciones profesionales, publica en Instagram y crea videos 
> automáticamente a partir de los datos de una propiedad.

🔗 **Plataforma:** [propiedades.toxirodigital.cloud](https://propiedades.toxirodigital.cloud)

---

## ¿Qué hace?

**Toxiro Propiedades** es una plataforma para agentes inmobiliarios que automatiza 
todo el proceso de publicación de propiedades — desde la carga de datos hasta 
la generación de contenido profesional y su publicación en redes sociales.

El agente ingresa los datos de la propiedad una sola vez. El sistema hace el resto.

---

## Funcionalidades principales

- ✅ **Generación de publicación profesional** — descripción optimizada con IA a partir 
  de los datos ingresados
- ✅ **Publicación automática en Instagram** — sin intervención manual
- ✅ **Generación de video** — contenido audiovisual de la propiedad generado automáticamente
- ✅ **Gestión de propiedades** — registro centralizado para seguimiento y gestión posterior
- ✅ **Panel del agente** — visualización y administración de toda la cartera de propiedades
